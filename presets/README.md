# Breath Lead - Preset Library

## Preset Format

JUCE XML preset format (.xml)

## Installation

### macOS:
- **Standalone**: Copy `.xml` files to `~/Library/Audio/Presets/SchillingerEcosystem/BreathLead/`
- **AU**: Copy to `/Library/Audio/Presets/SchillingerEcosystem/BreathLead/`
- **VST3**: Copy to `~/Library/Audio/Presets/SchillingerEcosystem/BreathLead/`

### DAWs:
Most DAWs will scan the user presets directory automatically.

## Preset Categories

### 1. Init Patches
- Golden Init Patch - Default, sounds good immediately

### 2. Breath Controllers
- Soft Breath - Gentle, intimate
- Medium Breath - Balanced, versatile
- Full Breath - Powerful, projecting

### 3. Tone Colors
- Dark - Warm, muted
- Bright - Clear, present
- Neutral - Balanced

### 4. Character
- Airy - Breath noise prominent
- Focused - Tight, controlled
- Open - Loose, resonant

### 5. Performance
- Subtle Vibrato - Gentle oscillation
- Wide Vibrato - Expressive wobble
- No Vibrato - Pure tone
